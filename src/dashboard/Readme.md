# Frontend - Acadify

## Overview

This is the frontend application for the Acadify platform.

It provides the user interface for:

- Administrators
- Teachers
- Students

## Technology

The frontend is built using **React**.

## Responsibilities

- Display dashboards and analytics
- Show student performance and reports
- Provide user-friendly interfaces for managing academic data

## Running the Frontend

```bash
pnpm --filter frontend dev
```

---

## Notes

This application communicates with the backend API and is part of the pnpm monorepo.
