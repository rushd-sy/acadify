# Backend - Acadify

## Overview

This is the backend service for the Acadify platform.

It is responsible for handling:

- Business logic
- Data management
- APIs used by the frontend

## Technology

The backend is built using **NestJS**, a scalable Node.js framework.

## Responsibilities

- Manage students, teachers, and administrators
- Handle academic data (reports, tests, results)
- Provide APIs for frontend consumption

## Running the Backend

```bash
pnpm --filter backend start:dev
```

---

## Notes

This service is part of a monorepo managed with pnpm workspaces.
